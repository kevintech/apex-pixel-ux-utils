# apex-pixel-ux-utils (formerly `orcl_apex_uiux_utilities`)

> A toolkit of UI/UX utilities, templates, and best practices for crafting pixel-perfect Oracle APEX applications, developed alongside the “Designer vs. Developer: Customizing Oracle APEX for Pixel-Perfect UI/UX” Kscope25 session.

---

## 📦 Features

- **Custom CSS snippets** for advanced theming and responsive layouts
- **JavaScript helpers** to streamline dynamic behaviors and animations
- **Illustrative templates** showcasing pixel-perfect component designs
- **Documentation** outlining best practices and usage guidelines

## 🚀 Getting Started

### Prerequisites

- Oracle APEX 24.2 or later
- Basic knowledge of HTML, CSS, and JavaScript
